# Kompare

A simple, fast JSON comparison tool with alphabetical sorting.

![Kompare Screenshot](https://via.placeholder.com/800x400?text=Kompare+Screenshot)

## Features

- **Compare JSONs** - Side by side comparison with diff highlighting
- **Sort A-Z** - Sort all keys alphabetically at all nested levels
- **Format** - Beautify JSON with proper indentation
- **Minify** - Compress JSON to single line
- **Dark/Light Mode** - Toggle between themes
- **Keyboard Shortcuts** - Quick actions

## Demo

[Live Demo](https://sayedabdulkarim.github.io/Kompare) *(coming soon)*

## Usage

1. Paste JSON in left panel
2. Paste JSON in right panel
3. Click **Compare** to see differences
4. Click **Sort A-Z** to sort keys alphabetically

| Shortcut | Action |
|----------|--------|

| `Ctrl/Cmd + Shift + S` | Sort A-Z |

- Zero dependencies
